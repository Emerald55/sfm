## Install
```bash
git clone https://github.com/Emerald55/sfm.git
cd sfm
mkdir build
cd build
cmake .. && sudo make install
sfm
```
Make sure to have __$EDITOR__, __$SHELL__, and __$PAGER__ environmental variables set.</br>
Press __'?'__ to get the list of keybindings.</br>
__--help__ for flags.
