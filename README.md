## Install
```bash
git clone https://github.com/Emerald55/sfm.git
cd sfm
mkdir build
cd build
cmake .. && sudo make install
sfm
```
Make sure to have $EDITOR, $SHELL, and $PAGER environmental variables set.</br>
Press '?' to get the list of keybindings.
